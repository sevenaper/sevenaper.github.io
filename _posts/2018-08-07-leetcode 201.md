这道题的思路就是把所有这个范围内的数字全部相与，不难发现最后得出的数字应该是这部分数字的公共部分。所以只要把最大值n和最小值m的左边公共的部分取出来，然后右边添0就可以。

```c++
class Solution {
public:
    int rangeBitwiseAnd(int m, int n) {
        int i = 0;
        while(m != n){
        	m = m >> 1;
        	n = n >> 1;
        	i++;
        }
        return m << i;
    }
};
```

我的思路如上面的代码所示，但是表现得不太好，看看别人速度最快的代码。

```c++
class Solution {
public:
    int rangeBitwiseAnd(int m, int n) {
        while (m < n) n &= (n - 1);
        return n;
    }
};
```

这段代码常数优化了很多，n &= (n - 1)这个操作很有意思，这相当于去掉最低位的1，可以给以后的题启发。
